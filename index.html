GitHub Pages 完全ガイド

HTMLファイルを無料で世界に公開する方法

所要時間: 約10-15分 • 初心者向け • 完全無料

GitHub Pagesとは？

GitHub Pagesは、GitHubが提供する完全無料の静的ウェブサイトホスティングサービスです。 HTMLファイルを簡単に世界中に公開できます。

完全無料・永続的
世界最大のプラットフォーム
高速・高信頼性
HTTPS対応

公開URLの形式

https://[ユーザー名].github.io/[リポジトリ名]/

例: https://tanaka123.github.io/my-website/

5つのステップで完了

1

GitHubアカウントを作成

  1. 1
    github.com にアクセス
  2. 2
    「Sign up」をクリック
  3. 3
    必要情報を入力
    • • メールアドレス
    • • パスワード
    • • ユーザー名（重要！）
  4. 4
    メール認証を完了

ユーザー名のコツ

  • • 短くて覚えやすい名前
  • • 英数字とハイフンのみ使用可能
  • • 公開URLに使われます
  • • 後から変更も可能
2

リポジトリを作成

  1. 1
    「New repository」をクリック
  2. 2
    Repository nameを入力
    例: my-website
  3. 3
    Publicを選択（重要！）
  4. 4
    「Add a README file」にチェック
  5. 5
    「Create repository」をクリック

重要なポイント

  • • Publicでないと公開できません
  • • README fileは設定を簡単にします
  • • リポジトリ名がURLに使われます
  • • 日本語名は避けましょう
3

HTMLファイルをアップロード

  1. 1
    「Add file」 → 「Upload files」
  2. 2
    HTMLファイルをドラッグ&ドロップ
  3. 3
    ファイル名を 「index.html」に変更
    （超重要！）
  4. 4
    「Commit changes」をクリック

ファイル名の重要性

必ず「index.html」にしてください！

❌ sample.html

❌ page.html

✅ index.html

index.htmlがWebサイトの「玄関」になります

4

GitHub Pages設定

  1. 1
    リポジトリの「Settings」タブをクリック
  2. 2
    左サイドバーの「Pages」をクリック
  3. 3
    Sourceで「Deploy from a branch」を選択
  4. 4
    Branchで「main」を選択
  5. 5
    「Save」をクリック

設定項目の説明

Deploy from a branch:
ブランチ（フォルダ）からWebサイトを作成
main:
メインのフォルダ（最新版）を使用
5

公開URL確認

  1. 1
    2-5分待つ（ビルド処理中）
  2. 2
    ページを更新（F5キー）
  3. 3
    青い成功メッセージを確認
  4. 4
    公開URLをクリックしてアクセス！

成功時の表示

✅ Your site is published at
https://username.github.io/repository-name/

このようなメッセージが表示されます

よくある問題と解決法

404エラーが表示される

原因:

  • • ファイル名がindex.html以外
  • • リポジトリがPrivate設定
  • • まだビルド中

解決法: ファイル名確認 → Public設定確認 → 時間を置いて再確認

URLが表示されない

原因:

  • • まだビルド処理中
  • • Pages設定が未完了

解決法: 5分待つ → ページ更新 → 設定再確認

ファイルを更新したい

  1. 1. リポジトリでindex.htmlをクリック
  2. 2. 「Edit this file」（鉛筆アイコン）
  3. 3. 内容を編集
  4. 4. 「Commit changes」
  5. 5. 2-3分で更新完了

カスタムドメインを使いたい

独自ドメイン（例: mysite.com）も設定可能です。

  • • Settings → Pages → Custom domain
  • • ドメインのDNS設定が必要
  • • 無料SSL証明書も自動発行
